# YouTube Music for Desktop
An unofficial desktop client for YouTube Music, with integrated lyrics from MusixMatch.

### Build requirements:
* Yarn package manager (1.10.0+)
* Node (v8.12.0+)
* Electron (v3.0.2+)

### Build guide
* Run `yarn run dist`
* Resulting artifacts are in the ./dist folder.
